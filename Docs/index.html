enclov-AI

Revolutionize your GitHub pull request workflow with AI-powered, automatic code reviews. enclov-AI analyzes your diffs, spots issues, and delivers expert-level feedback — all within GitHub.

View on GitHub

Features

  • • AI-powered review comments using OpenAI GPT models
  • • Seamless GitHub App integration via webhooks
  • • Detailed feedback on code diffs and PRs
  • • Built with FastAPI, Celery, Redis for scale
  • • One-line Docker Compose deployment

Get Started

1. Install the enclov-AI GitHub App on your repository.
2. Point the webhook to your deployed FastAPI server.
3. Open or update pull requests — AI handles the rest.
